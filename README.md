# Kintone Group Tabs Customization

Kintone アプリのフォーム画面において、グループフィールドをタブ形式で表示するカスタマイズです。  
グループ単位の切り替えを縦型タブ（Vertical Tabs）として実装し、ユーザーがフォーム内容を直感的に操作できるようにします。

---

## 📁 構成ファイル

| ファイル名 | 内容 |
|-------------|-------|
| `tab.js` | グループを自動検出してタブ化するメインスクリプト |
| `tab.css` | タブのスタイル（縦型UI）を定義するスタイルシート |

---

## 🧩 機能概要

- Kintoneフォーム内の各グループを自動的にタブとして変換  
- タブクリックでグループ単位の切り替え  
- 「hidden」を含むグループ名はタブを非表示  
- グループの初期表示は最初のタブ  
- 登録・編集・詳細画面すべてに対応  

---

## 💡 動作概要

1. 登録・編集・詳細画面イベントで処理を実行  
2. `.group-label-gaia` を持つグループ要素を検出し、グループ単位でタブを生成  
3. タブクリックで該当グループを表示、他のグループを非表示化  
4. グループ名に「hidden」を含む場合はタブを非表示  
5. 初期状態で最初のタブを選択し、フォームを表示  

---

## 🎨 スタイル仕様（tab.css）

| クラス名 | 説明 |
|-----------|------|
| `.vertical-tabs` | 通常タブ（縦書き・グレー背景） |
| `.vertical-tab-selected` | 選択中タブ（白背景） |
| `.vertical-tab-container` | 各グループのコンテナ |
| `.vertical-tab-container-hidden` | 非表示状態のグループ |
| `.border-less-group` | グループ枠線を非表示 |
| `#vertical-tab-container` | タブ全体の配置コンテナ |
| `#vertical-tab-container-wrapper` | 固定表示用のラッパー |
| `#record-gaia.show` | タブ生成後の表示制御 |

---

## ⚙️ 導入手順

1. Kintoneの管理画面で対象アプリを開く  
2. 「アプリの設定」→「JavaScript / CSSでカスタマイズ」を開く  
3. 以下のファイルを追加する  
   - JavaScript: `tab.js`  
   - CSS: `tab.css`  
4. 「PC用表示」にのみ適用  
5. アプリを再公開  

---

## 🚀 使用方法

1. フォームに複数のグループフィールドを配置  
2. 各グループのラベル名がそのままタブ名として表示される  
3. グループ名に「hidden」を含めると、そのタブは非表示  

---

## 🧭 表示例

| グループ名 | 表示状態 |
|-------------|-----------|
| 顧客情報 | タブ表示 |
| 契約情報 | タブ表示 |
| 内部設定(hidden) | 非表示 |

## ✨ 注意事項

- Kintone UI 構造の変更が行われた場合、動作に影響が出る可能性があります。  
- CSSの `writing-mode` を `horizontal-tb` に変更すれば、横型タブ表示にも対応可能です。
